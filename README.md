# Keroway Tap

Homebrew tap for [keroway](https://github.com/keroway) tools.

## Formulae

| Formula | Description |
|---------|-------------|
| [tdsl](Formula/tdsl.rb) | Timeline DSL compiler — text-based timelines with Wikidata import |

## How do I install these formulae?

```sh
brew install keroway/tap/tdsl
```

Or tap first, then install:

```sh
brew tap keroway/tap
brew install tdsl
```

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "keroway/tap"
brew "tdsl"
```
