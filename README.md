# Keroway Tap

## How do I install these formulae?

`brew install keroway/tap/<formula>`

Or `brew tap keroway/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "keroway/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
